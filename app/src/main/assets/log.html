LOGARITHM

The logarithm of a number N to the base 'a' is the exponent indicating the power to which the base 'a' must be raised to obtain the number N.

This is represented as loga(N).

  • logxN, read as log of N to the base x and ax = N. If base is equal to e, then it is given as ln(N) or loge(N).
  • Necessary logarithm conditions: N > 0, a > 0; a ≠ 1
  • loga(1) = 0
  • loga(a) = 1
  • log1/a(a) = -1
  • loga(1/a) = -1
  • elna x = ax
  • loga(xy) = loga(x) + loga(y) where x, y > 0
  • loga(x/y) = loga(x) - loga(y) where x, y > 0
  • loga(xr) = r ⋅ loga(x);     x > 0
  • logas(x) = 1/s ⋅ loga(x);     x > 0
  • loga(x) = 1⁄logx(a) ; x > 0, x = 1
  • loga(x) = logp(x)⁄logp(a) ; x > 0, a, p > 0, p ≠ 1, a ≠ 1
  • loga b ⋅ logb c ⋅ logc d = loga d ; a, b, c, d > 0, ≠ 1
  • a loga x = x ; a > 0, a ≠ 1
  • a logb c = c logb a ; a,b,c > 0 and b ≠ 1
  • loga(x) = loga(y)     ⟹     x = y;     x, y > 0;     a > 0, a ≠ 1
